Revised function scores (updated from initial research)
F-04 Intent cards (was: Remove → REVISED: Keep compressed) Revised ↑
Growlink is a 3-vertical marketplace — the intent selector IS the product identity. Keep as a compact 36px pill row, not large tiles. Auto-skip for returning users.
F-05 Skill chips (was: Static → REVISED: Intent-aware dynamic) Revised ↑
Chips must adapt to selected intent. 3 chip modes: all-categories (no intent), intent categories (intent selected), skill chips (category selected).
F-23 (NEW) Intent Navigation — 70% first-time users New · Critical
3-path entry (Get Help / Learn/Create / Work/Earn). Compact pill row on home. Tab bar in Discover. Auto-routes on search keywords.
F-24 (NEW) Hierarchical category browser — 45% per Discover session New · High
Main cat → subcategory → skills drill-down. Lives in Discover only. Home shows chips only — not the full browser.

One-line rule for every screen
Home Shows L1 (3 intent pills) + L2 (top 6 category chips) + popular skill counts
Discover Shows L1 (tab bar) + L2 (category chips) + L3 (subcategory grid) → drill to L4 providers
Search Shows all 4 levels in autocomplete. Routes to correct level on tap.
Filter Shows full hierarchy as collapsible accordion. Intent at top, skills nested inside.
